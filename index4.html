Vi är Code Cabin

Passionen för kreativ kodning ligger som grund till produkterna vi skapar. Idén grundades efter en föreläsning på Polacksbacken i slutet av Maj 2018. Att lägga programmering och utbildningen på hyllan under sommaruppehållet kändes ointressant.

Efter möten med brainstorming, gnutta tveksamhet och konsultation bestämmde vi oss för att skapa Code cabin. Målet är att lära oss utveckla web och skapa unika och proffsiga produkter. Arbetet sker på cafér, i våra studentlyor, på flygplan eller i sommarstugan.

Code cabin är din lokala programmerare, som finns där i ur och skur.

Morsing korsning, swipea höger!
